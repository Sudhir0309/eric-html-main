In a nutshell

The government of Ontario web apps are redesigned to facilitate specific user tasks, all these tasks leading to a buying to flow. Using Figma to create prototypes into which I’ve integrated the  Ontario Design System (ODS), I’ve collaborated with multiple stakeholders (other designers, developers, and product owners) to:

  • Simplifies the ordering process by breaking it into manageable steps to maintain user’s focus, reducing cognitive effort through streamlined business logic, and effectively handling both system and user errors to ensure a smooth user experience while discouraging hacking attempts.
  • Connects these sub-processes into a seamless experience, simplifies error handling by linking non-sensitive errors to specific steps making it easy for users to address, and streamlines the checkout process by prioritizing key information such as payment summaries and terms and agreements.
